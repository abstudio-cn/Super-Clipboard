# Super-Clipboard
input paste app for online web VNC like novnc

键盘模拟剪贴板程序，给类vnc类似于novnc提供模拟粘贴支持，除此之外正常粘贴功能依旧可以使用。

**本程序允许在您评估并规避第三方组件风险后进行商用，二次开发请遵循MIT协议，注明abstudio，感谢您的支持。**  

* * *

**超级剪贴板，一个基于WPF/C#可代替您打字的剪贴板。**  
  
**前情提要：**  
**某一天我正在使用某葵远程Q0.5系列的盒子的时候，恰好需要输入一段非常冗余的Linux安装软件的代码给我的新到的设备，可惜的是，没有直接远程粘贴的功能，于是这个项目就诞生了。**  

  
**一、这个软件能干什么（软件简介）**  
目前实现的功能有：  
**1、基础功能：复制文本/图片/软件，进行ctrl+v的普通粘贴以及ctrl+alt+v的模拟键盘输入**  

**2、历史记录，您可以查看曾经复制的粘贴内容，并且双击切换待粘贴的内容，对于懒得重复切换网页粘贴代码的您来说，简直是天堂。**  
  
**3、后台保留，您可以关掉窗口，不影响您的正常使用。**  

未来会实现的功能（画大饼）：

******·****个性化：支持白天夜间主题切换**  
****·******支持：在历史记录中增加对主流文件格式的预览支持，让您更直观的了解剪贴板中的一切**  
****·******个性化：自定义您的软件背景。**  
****·******支持：非主流的文件格式的浏览支持。**

**二、我凭什么要用它，它有什么优势吗（软件简介）**  
  
当您在使用KVM平台比如promox、某些虚拟化平台或者一些本地远控/监控器的平台时进行系统救援或者轻运维时，您可能会输入一段冗长的代码还可能会输错。  
这个软件可以替代您输入字符的过程，您只需轻松按下模拟输入的快捷键，即可轻松粘贴别人/AI问来的代码，**轻松实现运维！**  

**三、我该怎么用（使用说明）**  
  
1、请在下载软件后解压使用，**双击“双击我启动”即可。**  
      **不会解压的小白，**作者也贴心的给您准备了**“****小白请用这个.exe****”，您只需要双击打开，然后点击解压即可**。  
      怕我在快捷方式里下毒的大佬，自己在解压好的文件夹内找到“superClipboard.exe”。  
  
2、打开后软件**会自动申请管理员权限**，没错软件需要一定的输入驱动支持才可以实现模拟输入，**请同意**，如果您不同意软件将无法使用  
      还有这不是用户协议。。  
  
3、打开后您可能发现窗口没有出现，**不要惊慌！**请到系统托盘找找有没有一个**蓝色粘贴板搭配一个橙色小点的图标**，对就是这个软件。  
      您可以**双击打开**，或者**点击右键打开**。  
  
4、即使您不打开软件，**软件也开始了他的工作！**  
      您可以正常的使用**ctrl+v的普通粘贴**以及**ctrl+alt+v的模拟键盘输入的粘贴。**  
  
5、如果您需要退出，请**右键之前您找到的那个图标，点击退出**即可退出软件。
